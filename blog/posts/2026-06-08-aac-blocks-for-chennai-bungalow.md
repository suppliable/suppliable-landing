## Wall area for a typical 1500 sqft bungalow

For a single-storey Chennai bungalow, here's the typical wall breakdown:

| Wall type | Length (rough) | Height | Area |
|---|---|---|---|
| External perimeter (4 sides) | 160 ft | 10 ft | 1,600 sqft |
| Internal partitions (10 walls avg) | 280 ft | 10 ft | 2,800 sqft |
| **Total wall area (both sides counted)** | | | **4,400 sqft** |

But we count each wall once — both sides will share the same blocks. So actual wall surface needing blocks is **2,200 sqft**.

## External walls — 6-inch AAC blocks

External walls go up in 6-inch blocks for thermal performance and structural width.

**6" block face area** (the visible side): 600 mm × 200 mm = 0.12 m² = **1.29 sqft per block face**.

For the 1,600 sqft of external wall:

```
Raw blocks = 1,600 ÷ 1.29 = 1,240 blocks
With 5% wastage = 1,302
Round up: 1,310 6-inch blocks
```

## Internal walls — 4-inch AAC blocks

Internal partitions use thinner 4-inch blocks (cheaper, takes less floor space).

Same block face area (1.29 sqft) since the face dimensions are the same — only the thickness changes.

For the 1,800 sqft of internal walls (after subtracting external):

```
Raw blocks = 1,800 ÷ 1.29 = 1,395 blocks
With 5% wastage = 1,465
Round up: 1,470 4-inch blocks
```

## Total blocks: ~2,780

Round up to **2,800 blocks total** (1,310 × 6" + 1,470 × 4").

## Thin-bed adhesive — Ramco Block Fix

Critical: **never** use cement-sand mortar with AAC blocks. It kills the thermal performance and reduces bond strength by 40-50%. Always use thin-bed adhesive (Ramco Block Fix or MYK equivalent).

Rule of thumb: **0.4 kg adhesive per 6-inch block**, scales linearly with thickness.

```
6" blocks: 1,310 × 0.4 kg = 524 kg
4" blocks: 1,470 × 0.27 kg = 397 kg
Total: 921 kg → 24 × 40 kg bags
```

Round up to **25 bags** of Ramco Block Fix (40 kg each).
